# Cargoo

Plataforma y PWA para conectar viajeros con espacio libre en su coche y personas que quieren enviar paquetes entre Suiza y otros paises como Espana, Portugal, Serbia o Albania.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase

## Desarrollo local

```sh
npm install
npm run dev
```

La app arranca por defecto en `http://localhost:8080`.

## Build

```sh
npm run build
```

## Producto

Cargoo esta pensado para:

- Publicar viajes recurrentes en coche con espacio libre.
- Publicar solicitudes de envio entre particulares.
- Hacer match entre remitentes y viajeros con una experiencia mobile-first.
- Funcionar como PWA instalable.
